# Tutorial Cara Menjalankan dan Menginstall AI-Model

## Langkah 1: Installasi Dependensi

Pastikan Anda telah menginstall Python 3.9 atau yang lebih tinggi. Kemudian, Anda perlu menginstall dependensi yang dibutuhkan oleh AI-Model menggunakan perintah berikut:

- pip install -r requirements.txt
- uvicorn api.ai_service:app --host 0.0.0.0 --port 8000
